To Start

```
npm install
npm run dev
```

## Scope

- [x] Add important points to the roadmap
- [x] Add assignments w.r.t. training progress and current topic
- [ ] UX Review & Style enhancement
- [ ] Use GitHub actions for deployment.
- [ ] Code refactoring.
